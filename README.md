# Sprite-Flight

![Screenshot 1](screenshots/SpriteFlightCover.png)

My first game! Completed by following the Unity Sprite Flight tutorial. I added a few varations such as my own sprites and a start screen to make it my own.

🎮 **[Play in Browser on itch.io](https://lilahskye.itch.io/sprite-flight)**

🎮 **[Or on Unity Play](https://play.unity.com/api/v1/games/game/7f703164-e7eb-4c5f-94af-f2b706b4f146)**

---

## About

Dodge asteroids and survive for as long as possible. 

## Controls

| Input | Action |
|-------|--------|
| Mouse | Aim |
| Left Click / Space | Thrust |


## Features

- Start screen with play button 
- Restart button
- High score tracker

## Built With

- [Unity 6](https://unity.com/)
- C#
- Unity UI Toolkit

## Development

First game built to learn game development. I learned about collisions, scene transitions, particle systems, audio implementation, sprite rendering, buttons and the larger UI toolkit. 

If I spent more time on this tutorial I would have made further changes to how the obstacles are generated. Sometimes they start out too fast and strike the player before they have time to react. I would have tuned this further to make every gameplay feel more cohesive, while maintaining randomness in how the obstacles are generated.
